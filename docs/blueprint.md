# **App Name**: Superteam Connect Malaysia

## Core Features:

- Landing Page & Content Display: Showcase Superteam Malaysia's mission, impact statistics, upcoming events, featured members, partner ecosystem, testimonials, and FAQs on the public-facing landing page.
- Member Directory & Search: Provide a searchable talent directory of community members in a responsive grid layout, complete with skill-based filters and modal profile views displaying photos, names, titles, companies, skills, social links, and achievements.
- Luma Events Integration: Display upcoming and past events dynamically sourced from Luma, allowing users to view details and link to the Luma platform for full event listings.
- Admin CMS for Content Management: Enable non-technical administrators to Create, Read, Update, Delete (CRUD) and Publish content for members, events, partners, statistics, and testimonials via a dedicated admin dashboard.
- Secure Admin Authentication: Implement robust authentication for administrator access to the CMS dashboard using Supabase Auth, ensuring data security and controlled content management.
- AI-Powered Content Drafting Tool: Provide an AI tool within the admin CMS to assist with generating initial drafts for event descriptions or partner blurbs, helping non-technical admins quickly create compelling content for review.
- Advanced SEO & Performance: Ensure the platform is optimized for search engines with Next.js metadata API, OpenGraph tags, semantic HTML, structured layout, and a focus on Lighthouse scores above 90 through optimized images, lazy loading, and server components.

## Style Guidelines:

- A dark theme with a vibrant 'Solana gradient' aesthetic. Utilize Solana purple (#9945FF) for primary interactive elements and key branding, and Solana green (#14F195) for accents, success states, and subtle glowing effects, often blended in gradients. The background will be a deep, dark charcoal or near-black to make these colors pop.
- Headlines use 'Space Grotesk' (sans-serif) for a modern, tech-inspired aesthetic, while body text employs 'Inter' (sans-serif) for legibility and neutrality in longer content.
- Implement minimalist, geometric line icons that align with the Web3 design patterns. Icons should feature subtle gradients or glowing effects where appropriate to complement glassmorphism elements.
- Adopt a minimal and spacious layout with ample negative space to enhance content readability and visual focus. Incorporate 'glassmorphism' elements, such as frosted, semi-transparent backgrounds, for depth and a modern Web3 feel. The design must be fully responsive across desktop, tablet, and mobile breakpoints.
- Utilize Framer Motion for subtle and smooth animations, including dynamic section reveals on scroll, interactive card hover effects, engaging animated counters for impact statistics, and fluid page transitions to enhance the user experience.
